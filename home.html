Ejercicio de bases de datos relacionales

Welcome

Este sitio web presenta un ejercicio de creación de una base de datos relacional.

Las tecnologías utilizadas son el lenguaje SQL, el sistema de gestión de bases de datos MySQL, la herramienta visual de diseño de bases de datos MySQL Workbench, el lenguaje de programación R, y R Markdown.

A través del menu de la izquierda puede acceder a las secciones del proyecto: la sección Introducción proporciona una descripción del proyecto, y el video exlicativo describe las fases de desarrollo del proyecto.
